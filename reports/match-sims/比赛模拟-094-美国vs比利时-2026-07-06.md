## 3. 剧本推演（挂在蒙卡分布上）

**剧本 A · 互咬拖平进加时（≈28.8% 平局块：1-1 13.6 + 0-0 7.6 + 2-2 6.0）**：最厚单一剧本。中大球锚下 1-1 比 0-0 厚近一倍——本场的平局形态更像"互有攻守打成 1-1"而非死守闷平。进加时后 25/48/27 仍是均势、近半数加时再平 → 全场 13.8% 滚进点球，届时纯掷硬币（西雅图主场声浪 vs 比利时大赛点球经验，经验值取平 0.50）。

**剧本 B · 比利时质量差 1 球带走（净负 1 球块 18.5%：1-2 8.6 / 0-1 6.3 打头）**：比利时 37.4% 胜面里近一半是"只赢一球"。黄金一代余晖的技术控制 + 锋线个体质量，打美国高强度逼抢漏出的身后空当——赢，但赢得不宽。比利时赢 2+ 只有 18.9%，碾压不在主牌面。

**剧本 C · 东道主正胜（33.8%：2-1 8.1 / 1-0 5.9 / 2-0 5.5 打头）**：主场声浪 + 体能冲击是美国这侧的核心变量，2-1 是美国胜面的模态格。美国净胜 2+ 仅 16.4%——大胜剧本同样是薄尾。

## 4. 靠谱度 + 缺口 + 重跑提示

- **靠谱度：中。** 加分项：三源幅度高度共振（极差 ≤3.2pp）、蒙卡拟合残差极小、让球三档与竞彩重合（±2.1pp）。扣分项：① **方向性分歧未消解**——共识站美国、Poly 站比利时，主锚押 Poly 一侧；若 sharp 错向，美/比两端概率要整体镜像 ±1.7pp 左右，晋级 52.1/47.9 可能翻面；② 大小球是竞彩单源软锚（返还 79.7%），p_over=0.55 含 −3.6pp 人工修正；③ 缓存仅 7/4 单日窗口，无漂移历史可交叉；④ pen_home=0.50 是经验值非市场价。
