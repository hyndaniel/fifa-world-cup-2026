## 一、总体结论

三份报告在大方向上高度一致：最稳的出线队集中在西班牙、阿根廷、英格兰、法国、巴西、德国、葡萄牙、比利时、墨西哥等传统强队。真正的差异主要不在豪门是否出线，而在几个小组的第二名和强第三名判断。

最核心的共识是：

- 西班牙、阿根廷、英格兰、法国、巴西基本都被视为小组赛最稳档。
- C组、H组、I组、J组、K组、L组的头部结构比较清楚。
- D组、E组、F组、G组和L组第三名竞争，是三份报告分歧最集中的区域。
- 2026赛制中，每组前二加8个最佳第三名晋级，因此“晋级32强概率”不能简单等同于“前二概率”。

## 二、三份报告的特点

| 报告 | 特点 | 优点 | 局限 |
|---|---|---|---|
| Fable-5 UltraCode | 最短，偏结论版 | 直接、易读，给出每组晋级排序和72场胜平负概率 | 没有拆分小组第一、前二、第三和晋级32强概率 |
| GPT-5.5 Deep Search Web | 最完整的研究型报告 | 明确使用 Elo、身价、伤停、赛程、气候、Poisson 和 200,000 次模拟 | 模型较重，对部分输入假设依赖更高 |
| GPT-5.5 Codex 超高 | 完整概率表加风格判断 | 对主场、对抗风格、战术匹配和第三名路径更敏感 | 部分边缘队概率与 Web 报告差异较大 |

综合来看，`gpt-5-5-web-deep-research-report.md` 的方法论最清楚，适合作为主参考；`fable-5-ultra-report.md` 适合作为快速结论；`gpt-5-5-codex-deep-research-report.md` 更适合观察主场红利、风格克制和强第三名路径。

## 四、关键分歧详解

### 1. D组：美国、土耳其、巴拉圭排序差异最大

D组是三份报告中分歧最明显的小组之一。

- Fable：美国 77 / 土耳其 73 / 巴拉圭 64 / 澳大利亚 59
- Web：土耳其 80.73 / 巴拉圭 68.29 / 美国 65.42 / 澳大利亚 57.26
- Codex：美国 87.9 / 土耳其 78.2 / 巴拉圭 60.6 / 澳大利亚 44.1

差异核心在于是否高度认可美国主场红利。Web 报告更看重土耳其和巴拉圭的基础强度，Codex 则显著抬高美国主场、运动能力和赛程优势。

观察重点：美国 vs 巴拉圭、美国 vs 土耳其。

### 2. E组：厄瓜多尔 vs 科特迪瓦

E组德国第一较稳，但第二名判断有分歧。

- Fable/Web：德国、厄瓜多尔、科特迪瓦、库拉索
- Codex：德国、科特迪瓦、厄瓜多尔、库拉索

Web 报告更看重厄瓜多尔的 Elo、南美出线质量和防守组织；Codex 更强调科特迪瓦的身体冲击、个体资源和冷门能力。

观察重点：科特迪瓦 vs 厄瓜多尔。

### 3. F组：日本 vs 瑞典

F组荷兰第一、突尼斯第四较一致，争议在日本和瑞典。

- Fable/Web：日本第二、瑞典第三
- Codex：瑞典第二、日本第三

Fable 和 Web 更认可日本的整体技术、节奏和稳定拿分能力；Codex 更重视瑞典的身体、传中、定位球和对日本的风格克制。

观察重点：日本 vs 瑞典。

### 4. G组：埃及 vs 伊朗

比利时被三份报告一致看作小组第一热门，但第二名存在分歧。

- Fable/Codex：埃及第二、伊朗第三
- Web：伊朗第二、埃及第三

Web 报告更重视伊朗的 Elo 和比赛纪律，但也提到伊朗后勤扰动；Fable/Codex 更看重埃及的萨拉赫体系和进攻决定性。

观察重点：埃及 vs 伊朗。

### 5. L组：加纳 vs 巴拿马是最大反向分歧

L组头两名英格兰、克罗地亚基本无争议，但第三名判断完全分裂。

- Fable：加纳 50 / 巴拿马 40
- Web：巴拿马 57.00 / 加纳 28.49
- Codex：加纳 57.9 / 巴拿马 28.6

这是三份报告中最典型的反向判断。Web 明显高看巴拿马的第三名路径，Codex 和 Fable 明显高看加纳的身体与前场冲击。

观察重点：加纳 vs 巴拿马。

## 六、综合采信建议

如果要形成一个综合版判断，可以采用以下口径：

1. 以 Web 报告作为主模型，因为它的方法说明最完整，并且区分了前二概率和晋级32强概率。
2. 对主办国球队，适当参考 Codex 报告，因为它对主场、旅行和风格匹配的处理更积极。
3. 对第三名边缘队，不要只看单一报告，尤其是 D组、E组、F组、G组、L组。
4. 重点关注直接对话，因为多数分歧会在一场比赛中被放大或消解。

## 八、最终判断

三份报告并不是互相否定，而是侧重点不同：

- Fable 是“快速赛前结论版”。
- Web 是“模型研究版”。
- Codex 是“战术与路径敏感版”。

若只给一个综合判断：豪门出线结论基本可靠；真正需要谨慎的是 D组美国、E组科特迪瓦/厄瓜多尔、F组日本/瑞典、G组埃及/伊朗、L组加纳/巴拿马。这些位置的预测分歧最大，也最可能产生实际冷门。
