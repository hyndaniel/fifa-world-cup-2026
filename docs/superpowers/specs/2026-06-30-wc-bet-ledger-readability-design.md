# wc-bet 下注复盘 可读性改版 · 设计

**日期**：2026-06-30　**对象**：`reports/agents/wc-bet__下注复盘.md`（661 行，6.24→6.30 共 13 节）
**目标**：整份替换为「结论先行 + 可扫读」格式。读者＝用户本人（懂术语）。**保留全部数字与真分析**，只重排呈现、砍散文墙。

## 设计原则（4 条）

1. **每节标题自带状态**：`## 日期 · 场次 · 🚦档 · 下/不下`（或票节：`· 全-EV 🎲`）。
2. **结论一句话置顶**：每节/每场开头一条 `> 一句话` 大白话结论 + 一条 `> 赛后` 实际兑现。
3. **赛前用表不用散文**：价值/票面信息进紧凑表，删掉重复的连续段落。
4. **赛后 ✅❌ 并进同一张表**（不另起「实际」长段）；深层逐腿明细用 `<details>` 折叠，scannable 在外、细节一键展开。

## 全局结构

1. **顶部** `# 盘口价值 · 推荐腿复盘` + 两行 disclaimer（保留）。
2. **名词&黑话**：保留全部条目（自读参照），但折进 `<details>` 或挪到文末，不占开头视线。
3. **正文**：13 节按时间序，各按 TYPE A/B 模板重排。**所有日期节必须在场**（不删节）。
4. **每节末尾的「数据镜像/非投注建议」尾巴**：压成一行小字。

## 红线（验收门槛）

- ✅ 所有 13 个日期节都在、顺序不变。
- ✅ 每条腿的 **赛后 ✅❌ / 实际盈亏 / 命中率** 一个不丢（赛后结果是台账核心资产）。
- ✅ 所有 value / EV% / 赔率 / Poly去水 数字保留（自读要自核）。
- ✅ best_leg、分档、软锚警示、整数让球走盘陷阱等**判断性结论**保留（这是报告的智力价值）。

- ❌ 不新增/不篡改任何赛果或数字；纯重排。
